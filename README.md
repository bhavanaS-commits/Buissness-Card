# Buissness-Card
A responsive digital business card website created using HTML and CSS to display personal profile and contact information. Clean and simple business card web project designed with HTML, CSS, and basic UI styling.

## 🚀 Features
- Responsive design (works on mobile & desktop)
- Clean and simple UI
- Displays profile photo and contact information

## 🛠️ Technologies Used
- HTML
- CSS
